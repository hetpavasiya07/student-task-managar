Project Overview
This project is developed using Flutter as part of academic coursework. It implements a simple Student Task Manager application with REST API integration.
The application allows users to manage tasks by adding, deleting, and marking tasks as completed. It also demonstrates how to fetch data from a public REST API, parse JSON data, and display it dynamically using ListView.

Objectives
The main objectives of this project are:
* To implement task management functionality using Flutter
* To integrate a public REST API into a Flutter application
* To parse JSON data using Dart
* To display dynamic data using ListView
* To demonstrate basic state management using StatefulWidget

Technologies Used
* Flutter
* Dart
* HTTP package
* JSONPlaceholder REST API

Features
* Fetch tasks from public REST API
* Parse JSON data
* Display tasks using ListView
* Add new tasks locally
* Delete tasks
* Mark tasks as completed
* Loading indicator during API fetch

API Used
The project uses the following public REST API for demonstration:
[https://jsonplaceholder.typicode.com/todos](https://jsonplaceholder.typicode.com/todos)
The API returns JSON data representing tasks, which are parsed and displayed in the application.

How to Run
1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Run the following command to install dependencies:

    flutter pub get

5. Run the application using:

   flutter run

Conclusion
This project demonstrates the implementation of a task management system along with REST API integration in Flutter. It highlights concepts such as JSON parsing, HTTP requests, dynamic UI updates, and state management.
